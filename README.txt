GITHUB PAGES QUICK START

1. Create a free GitHub account.
2. Create a new repository named chord-trainer.
3. Upload ALL files from this package.
4. In repository Settings -> Pages.
5. Source: Deploy from branch.
6. Branch: main / root.
7. Save.
8. Wait ~1 minute.
9. Visit the URL GitHub gives you.
10. Open in Safari on iPhone.
11. Share -> Add to Home Screen.
